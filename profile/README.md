# Warehouse Management & B2B Ordering Platform

A unified platform for managing products, inventory, orders, payments, and financial reporting for B2B operations. This system provides a single source of truth for warehouse operations and integrates with a third-party payment gateway for transaction processing. This project is designed as an end-to-end solution covering:

- Product and inventory management
- B2B ordering workflow
- Role-based business dashboard
- Payment processing via external gateway
- Financial and transaction reporting
